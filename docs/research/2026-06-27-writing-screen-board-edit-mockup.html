집필 화면에서 보드 보고·수정 — 레이아웃 방법

현재: 집필 화면 우측 보드 목록의 "열기"를 누르면 보드 페이지로 완전 이탈(돌아오려면 집필 메뉴→작품 재진입). 셋 다 "열기"를 인라인으로 돌려 집필 화면을 유지하는 게 공통이고, 보드를 어느 정도 넓이로 펼치느냐만 다릅니다. 셋 다 보드는 편집 가능(카드 생성·잇기·종류·삭제 = 보드 페이지와 동일 캔버스).

※ 인라인 편집 패널 자체는 이미 있음(상단 "보드 참조" 버튼 → 043 슬라이드오버). 차이는 폭/배치뿐. 마음에 드는 안을 골라 주세요.

A · 오버레이(우측 슬라이드오버) A

지금 자산 그대로. 보드 클릭 → 우측 ~40%로 보드가 덮여 펼쳐짐(편집). ✕/ESC로 닫으면 원고 그대로. 변경 가장 작음 · 보드 열면 원고 옆 보기는 좁음

공통(셋 다)

  • 집필 화면 우측 보드 목록 "열기" → 완전 이탈 대신 인라인으로(그 보드 preselect). 집필 화면 유지.
  • 인라인 보드 = 편집 가능(카드 생성·잇기·종류·삭제·드래그 = 보드 페이지와 동일 캔버스).
  • 보드 페이지로 진짜 이동이 필요하면 작은 "↗ 전체 화면/보드 페이지" 액션은 남김.
  • FE 중심(BE 변경 0 — 기존 reference 패널·캔버스 재사용).
