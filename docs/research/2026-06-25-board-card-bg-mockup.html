보드 카드 배경 틴트 — 색상 비교

현재 카드는 흰 배경 + 좌측 색 띠(border-left)로만 타입을 구분한다. 요청대로 배경색까지 타입 색으로 통일(Miro 스타일)하되 채도를 낮춰 편안하게. 좌측 띠는 떼고 배경 틴트 + 같은 계열 전체 테두리로 바꾼다(이게 시각적으로도 더 또렷하고, 좌측 띠 패턴 자체가 디자인 안티패턴이라 정리됨).

5종 타입 = 플롯·인물·장소·테마·메모 · 색은 기존 팔레트(terracotta·teal·emerald·violet·slate) 유지 · 저채도 3단계를 비교 · 본문 대비 ≥ 4.5:1(AA)

아래 현행을 기준으로 A 옅게 → B 중간(권장) → C 또렷 순으로 배경이 진해진다. 같은 dotted 캔버스 위에 5종을 나란히 둬서 ① 타입 구분이 충분한지 ② 채도가 불편하지 않은지 한 화면에서 본다. 마음에 드는 단계(또는 "A와 B 사이" 같은 중간)를 골라주면 그 값으로 실제 카드(CardNode)에 반영한다.

기본 추천은 B(중간) — 소설비의 "절제된 따뜻함" 원칙(저채도·계열색·full border)에 가장 맞고, 타입 구분과 편안함의 균형이 가장 좋다.
