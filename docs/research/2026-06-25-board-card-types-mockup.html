트랙 D — 카드 종류 4종 정합 + 종류 선택 칩 UI

확정된 결정: 종류 4종(인물·장소·사건·테마) · 기본 무지정(종류 없음) · 생성 후 칩으로 부여·재탭 해제. 색은 트랙 B 에서 채택한 저채도 팔레트(−50 틴트) 그대로. 보드는 colorMode=light 고정.
이 목업에서 고를 것 = ① 무지정 카드의 외관, ② 종류 선택 칩("이건 뭔가요?")의 위치.

① 4종 종류 색 (확정 제안 — 새 색 추가 0)

인물=teal · 장소=emerald · 사건=terracotta · 테마=violet. '사건'은 폐기되는 'plot'의 terracotta를 그대로 승계(plot≒event 의미 동일). 'note(메모)'의 slate 회색은 아래 ②의 무지정 카드 전용으로 옮긴다. 즉 가짓수만 5→4, 새 색은 안 만든다.

② 무지정 카드(종류 없음)의 외관 — A / B / C

생성 직후 기본값 = 종류 없음. "아직 종류를 안 정한 카드"를 어떻게 보일지. 4종 틴트와 구분되면서, 종류를 강요하지 않는 "중립" 느낌이 목표.

A. 중립 회색 채움 추천

4종과 같은 "옅은 틴트" 문법을 유지하되 회색(slate-50). 종류 배지 자리에 옅은 "종류 없음".

종류 없음
갑자기 떠오른 반전 아이디어
일관성 최고. 4종이 전부 "옅은 틴트 카드"라 무지정도 같은 문법(회색 틴트). 회색 = 중립의 자연스러운 신호.

B. 흰 배경 + 회색 점선

점선 테두리로 "미정" 상태를 적극 암시. 배지 없음.

갑자기 떠오른 반전 아이디어
"아직 안 정함"이 가장 또렷. 다만 점선은 "임시/미완"으로도 읽혀, 끝까지 무지정으로 쓸 메모 카드엔 과한 신호일 수 있음.

C. 흰 배경 + 옅은 실선

가장 미니멀. 평범한 흰 카드.

갑자기 떠오른 반전 아이디어
깔끔하지만 캔버스 배경(흰색)과 대비가 약해 카드 경계가 흐릿. 4종 틴트 카드들 사이에서 "비어 보임".

③ 종류 선택 칩("이건 뭔가요?")의 위치 — A / B / C

카드를 선택하면(클릭) 종류 선택 칩 4개가 나타난다. 한 칩 탭 = 색/배지 즉시, 같은 칩 재탭 = 무지정 해제. 이미 카드 선택 시 하단에 ↗ 연결할 카드 고르기(잇기) 버튼이 있어 둘의 공존이 관건.

A. 카드 아래 분리 트레이 추천
