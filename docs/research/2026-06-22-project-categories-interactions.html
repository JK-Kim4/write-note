작품을 폴더로 옮기는 방식, 3가지를 직접 끌어보세요

카드를 마우스로 집어서 폴더로 끌어다 놓으면 분류됩니다. 카드의 ⋯ 메뉴로도 옮길 수 있어요(터치·키보드 대안). 세 방식은 같은 데이터를 공유하니, 한 방식에서 옮긴 작품이 다른 탭에도 그대로 반영됩니다. 마음에 드는 인터랙션을 고르면 그걸로 실제 구현을 설계할게요.

탐색용 정적 프로토타입입니다 (production React 코드 아님). 드래그 엔진은 포인터 기반 — 데스크탑 마우스 기준. 실제 구현에서는 터치/키보드 접근성, React Query 캐시 갱신, 서버 저장이 더해집니다.
