# 소설비(Soseolbi) v0.5.0 릴리즈 노트

날짜: 2026-07-01

여러 보드에 흩어진 메모 카드를 한곳에서 관리하고, **집필하면서 곁에 두고 참조**할 수 있는 *카드 관리*를 새로 선보입니다.

## ✨ 새 기능

### 카드 관리

- 보드 화면에 **[보드 | 카드] 탭**이 생겼습니다. [카드] 탭에서 **여러 보드에 흩어진 카드**를 한 목록으로 모아 봅니다.
- 각 카드가 **어느 보드에 속하는지**(작품·시리즈) 함께 표시되고, 어느 보드에도 없는 **독립 카드**도 여기서 관리합니다.
- **검색**(내용·보드 이름)과 **필터**(소속: 전체/보드/독립, 종류: 인물·장소·사건·주제·무지정)로 원하는 카드를 빠르게 찾습니다.
- 보드 없이 쓰는 **독립 카드(간단한 메모)를 바로 만들 수 있습니다.**
- 카드를 열어 **종류·내용을 고치고**, **삭제**하거나(다른 카드와 연결돼 있으면 미리 경고), **소속 보드를 옮길** 수 있습니다.

### 집필하면서 카드 참조

- 집필 화면의 **"보드 참조"** 패널에 **[카드] 뷰**가 추가됐습니다. 쓰던 자리를 떠나지 않고 카드를 곁눈질합니다.
- 그 작품과 관련된 카드를 **이 작품 보드 → 시리즈 보드 → 독립 카드** 순서로 묶어 보여주고, **검색**으로 좁힐 수 있습니다.
- 카드를 열면 종류·내용을 바로 확인할 수 있습니다(참조 전용).

## 🔧 개선

- 참조 패널의 **[보드 | 카드] 전환**이 한눈에 들어오도록 **탭 모양으로 다듬었습니다**(🕸 보드 / 🗂 카드).
- 참조할 보드를 고를 때 **작품 보드 / 시리즈 보드로 구분**해 보여줍니다.
- 집필 화면 오른쪽 **보드 패널의 접기 화살표를 정리**해 불필요한 혼란을 없앴습니다.
