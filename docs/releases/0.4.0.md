# 소설비(Soseolbi) v0.4.0 릴리즈 노트

날짜: 2026-06-28

써둔 글을 **링크 하나로 공유**하고, 읽은 사람에게서 **그 자리에 피드백**을 받을 수 있는 *공유하기*를 새로 선보입니다.

## ✨ 새 기능

### 공유하기

- 작품이나 시리즈를 **공유 링크**로 내보내면, 로그인하지 않은 사람도 링크만으로 읽을 수 있습니다.
- 공유한 **그 시점의 내용으로 고정**됩니다(불변 스냅샷). 이후 원고를 고쳐도 공유본은 그대로라, 어느 시점의 글을 보여줬는지 분명합니다.
- 링크를 받은 **로그인 회원은 본문 구간을 짚어 피드백**을 남길 수 있고, 이 피드백은 **작가에게만** 보입니다(다른 열람자에게는 비공개).

### 공유, 더 쉽게 (사용성 개선)

- 헤더 최상위에 **"공유"** 메뉴가 생겼습니다.
- 작품 카드·시리즈 타일에서 **바로 공유**할 수 있고, 공유 중이면 **"● 공유 중 · N"** 으로 표시됩니다.
- **받은 피드백 읽음 관리** — 안 읽은 피드백 개수를 보여주고, 열어보면 읽음으로 정리됩니다.
- 링크 **삭제**와 작품·시리즈당 **최대 5개** 링크 관리.

## 🔧 개선·버그 수정

- **공유 창 닫기(✕)가 안 되던 문제 수정** — 공유 창이 작품 카드와 겹쳐, 누르는 순간 창이 흔들려 닫기 버튼이 빗나가던 문제를 바로잡았습니다.
- **삭제한 작품의 "받은 피드백" 배지가 사라지지 않던 문제 해소.**
- 공유한 작품을 **공개 해제할 때, 그동안 받은 피드백도 함께 사라진다는 경고**를 추가했습니다.

## 🗂 정리

- **데스크톱 앱을 정리**하고 웹 앱 전용으로 전환했습니다 — 소설비는 브라우저에서 그대로 쓰실 수 있습니다.
