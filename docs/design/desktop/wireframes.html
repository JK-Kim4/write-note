Focus Studio Wireframes — Desktop MVP

Desktop MVP의 구현용 와이어프레임입니다. 디자인 방향은 Focus Studio 하나로 부릅니다 — 조용한 macOS native 톤(시각)과 글쓰기 canvas 중심 레이아웃(구조)을 동시에 뜻합니다. 목적은 “예쁜 화면”보다 구현 중 흔들리지 않을 화면 구조와 톤을 고정하는 것입니다. 첫 MVP는 일반 에디터, 작품, 빠른 메모, 작품 연결 메모만 포함합니다. 상세 token은 01-design-system.md, 상태 정의는 03-screen-states.md를 따릅니다.
