# 03 · 화면 상태 (Screen States)

> empty / loading / error / saving / focus 상태 정의. 각 상태는 최소 1개 화면에서 구체화한다.
> 색/형태 token은 [`01-design-system.md`](./01-design-system.md).

## 상태 × 화면 매트릭스

| 상태 | 주 정의 화면 | 적용 화면 |
|---|---|---|
| empty | 메모함 / 연결된 메모 | 작품, 메모함, 집필실 panel |
| loading | 작품 | 전 화면 |
| error | 빠른 메모 / 집필실(저장) | 저장·생성 동작 |
| saving | 집필실 | 집필실, 빠른 메모 |
| focus | 전 화면 공통 (input/카드) | 전 화면 |

---

## 1. Empty

**원칙:** 빈 상태는 **조용하게**. 큰 일러스트·과장된 안내 금지. 한 줄 안내 + (있다면) 단일 액션.

- **작품 — 작품 목록 비었을 때:** 우측 리스트 자리에 dashed 카드 1개, `--faint` 텍스트 "아직 작품이 없다. 왼쪽에서 첫 작품을 만든다."
- **집필실 — 연결된 메모 비었을 때:** dashed empty 카드 — "작품에 연결된 메모가 없으면 이 패널은 조용히 비어 있다." (목업에 반영됨)
- **메모함 비었을 때:** dashed 카드 1개 — "캡처한 메모가 여기 모인다. [빠른 메모]로 시작한다."
- **에디터 본문 비었을 때:** placeholder 한 줄(`--faint`) — "여기에 쓴다." 그 외 장식 없음.

## 2. Loading

**원칙:** spinner 남발 금지. 로컬 저장이라 대부분 즉시. 지연 가능 지점만 **얕은 skeleton**.

- **작품 진입 시 — 작품 목록 로딩:** 카드 자리에 회색 skeleton 막대(`--line` 톤) 2~3개. blue 사용 안 함.
- **집필실 문서 로딩:** paper 영역에 prose-line skeleton 3~4줄.
- skeleton은 정적 회색. 펄스 애니메이션은 최소(없어도 됨).

## 3. Error

**원칙:** 에러는 **현재 위치에서 짧게**. 전역 toast 폭주 금지. `--danger` 텍스트 한 줄 + 재시도 동선.

- **빠른 메모 저장 실패:** modal 하단 저장 버튼 옆에 `--danger` 한 줄 "저장하지 못했다. 다시 시도." + 입력 내용 유지(소실 금지).
- **집필실 자동저장 실패:** titlebar 우측 상태 라벨이 `저장됨` → `저장 실패`(`--danger`)로 바뀌고, 다음 자동저장 시 재시도. 본문 내용은 절대 잃지 않는다.
- **작품 생성 실패:** 새 작품 블럭 액션 아래 `--danger` 한 줄. 입력값 유지.

## 4. Saving (자동저장 상태)

**원칙:** 집필실의 핵심 상태. titlebar 우측 라벨로 **조용히** 표현. 모달·배너 금지.

| 단계 | titlebar 우측 라벨 | 비고 |
|---|---|---|
| 변경 없음/저장 완료 | `저장됨 · 1,248자` | 기본 상태(`--muted`). |
| 입력 중 (debounce 대기) | `저장 중…` | 짧게. |
| 저장 실패 | `저장 실패` | `--danger`, 다음 입력/주기에 재시도. |

- 글자수는 항상 함께 표기.
- "저장됨"은 blue 아님 — `--muted` 회색. 저장은 정상 상태이므로 강조하지 않는다.

## 5. Focus

**원칙:** focus는 **blue가 허용되는 두 자리 중 하나**(다른 하나는 primary action). 키보드 동선이 또렷해야 한다.

- **input/textarea focus:** 테두리 `--accent`(#3b82f6), 아주 옅은 focus ring(`--accent-soft` 계열). 배경은 흰색 유지.
- **선택된 카드(작품 목록 활성):** 테두리 `--accent-border`(#93c5fd) + 배경 `--accent-bg-faint`(#f8fbff).
- **활성 rail 버튼:** 텍스트 `--accent-strong`(#1d4ed8) + 배경 `--accent-soft`(#dbeafe).
- **에디터 focus:** 별도 강조 없음 — 에디터는 항상 주인공이라 추가 chrome 불필요. 커서만으로 충분.

> focus 외에 hover 강조는 최소. 클릭 가능한 카드만 미세한 배경 변화(`--panel-2`) 정도.
