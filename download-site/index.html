소설비 다운로드

컨텍스트가 안 죽는 작가용 작업공간. Windows 와 macOS 에서 사용할 수 있습니다.

Windows용 다운로드 .exe · Windows 10 이상 macOS용 다운로드 .dmg · Intel · Apple Silicon

설치 방법

아직 코드 서명 전이라, 처음 실행할 때 운영체제가 "확인되지 않은 앱" 경고를 보여줍니다. 아래 단계를 따르면 안전하게 설치할 수 있습니다.

Windows

  1. 1.내려받은 Soseolbi-Setup.exe 를 실행합니다.
  2. 2."Windows의 PC를 보호했습니다" 창이 뜨면 → 추가 정보 → 실행 을 누릅니다.
  3. 3.설치가 끝납니다 (관리자 권한이 필요 없습니다).

macOS

  1. 1.내려받은 Soseolbi.dmg 를 열고, 소설비를 응용 프로그램 폴더로 드래그합니다.
  2. 2.앱을 처음 열면 "확인할 수 없어 열 수 없습니다" 안내가 나옵니다 → 완료 를 누릅니다.
  3. 3.시스템 설정 → 개인정보 보호 및 보안 으로 가서, 아래쪽 "확인 없이 열기" 를 누르고 암호를 입력합니다.
  4. 4.실행 중 "키체인에서 … 사용하려고 합니다" 창이 뜨면, Mac 로그인 암호를 입력하고 허용 을 누릅니다.
  5. 5.이후에는 평소처럼 바로 열립니다 (이 과정은 처음 한 번, 새 버전 업데이트 시 다시 나올 수 있습니다).
문제가 있으면 릴리스 페이지에서 직접 받을 수 있습니다.
