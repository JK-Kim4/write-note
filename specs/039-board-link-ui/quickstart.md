# Quickstart — 연결(Link) UI dogfooding 게이트 (트랙 A)

> 캔버스 상호작용(핸들 hover·드래그 선·초록 강조·빈곳 drop 모달·잇기 모드·dim·hover ✕)은 jsdom 미검증(research §검증 한계). 아래는 **사용자 dogfooding 게이트** — 자동 게이트(typecheck·lint·test·build) GREEN 위에 얹는다.

## 사전: 로컬 풀스택 기동 (메모리 [[local-dogfooding-needs-backend]])

```bash
docker compose up -d --wait postgres
cd backend && ./gradlew bootRun --args='--spring.profiles.active=local'   # 별 터미널
cd frontend && pnpm dev                                                    # → http://localhost:3000
```
로그인 후 `/boards`에서 보드 1개 진입(카드 2장 이상 준비).

## 자동 게이트 (먼저 GREEN)

```bash
cd frontend && pnpm typecheck && pnpm lint && pnpm test && pnpm build
```
- `linkGraph.test.ts`(어댑터·canLink·이웃집합) GREEN.
- `pnpm build` = RSC 경계(`"use client"`) 검출.

## dogfooding 시나리오 (전부 통과 = 게이트 닫힘)

### US1 카드 잇기 (P1)
- [ ] **핸들 노출**: 카드에 hover하면 가장자리 연결점이 나타나고, hover를 떼면 사라진다(평상시 안 보임).
- [ ] **드래그 잇기**: 연결점을 끌면 선이 커서를 따라오고, 유효한 다른 카드 위에서 그 카드가 초록 강조 → drop하면 두 카드가 선으로 이어진다.
- [ ] **빈 곳 drop**: 연결점을 빈 곳에 놓으면 "여기에 새 카드를 만들어 이을까요?"가 뜬다. "만들기" → 그 자리에 새 카드(기본 종류=플롯/사건) 생기고 출발 카드와 이어진다. "취소" → 새 카드·연결 안 생김.
- [ ] **클릭-클릭**: 카드 선택 → "잇기" 버튼 → "이을 카드를 선택하세요" 안내 → 다른 카드 클릭 → 이어진다. ESC/빈 곳 클릭으로 취소.
- [ ] **중복 가드**: 이미 이어진 A·B를 다시 이으려 하면(어느 방향이든) 새 선이 안 생긴다(에러 토스트도 안 뜸).
- [ ] **자기연결 가드**: 한 카드의 연결점을 자기 자신에 놓으면 아무 일도 안 일어난다.
- [ ] **무방향**: 연결선에 화살표가 없다.

### US2 연결 끊기 (P2)
- [ ] **hover ✕**: 연결선에 hover하면 가운데 ✕가 뜨고, 누르면 연결이 끊긴다.
- [ ] (보조) 연결선 클릭 선택 후 Delete 키로도 끊긴다.
- [ ] **영속**: 끊은 뒤 보드를 다시 열어도 끊긴 상태 유지.

### US3 이웃 하이라이트 (P3)
- [ ] **강조**: 여러 카드와 이어진 카드를 선택하면 그 카드·이어진 카드·잇는 선만 또렷하고 나머지는 흐려진다.
- [ ] **복원**: 빈 곳을 클릭하면 전부 원래 또렷함으로 복원.
- [ ] **공존**: 하이라이트 중에도 카드 본문 더블클릭 편집·드래그 배치 정상.

### 영속·롤백
- [ ] **재진입 복원**: 맺은 연결이 보드 재진입 시 100% 복원(누락·유령 0).
- [ ] **실패 롤백**(가능 시): 연결/끊기 저장 실패 시 화면이 직전 상태로 복원되고 알림이 뜬다.

### 회귀 (SC-007)
- [ ] 기존 카드 만들기(+노드)·본문 편집·드래그 배치·뷰포트 복원이 그대로 동작.
- [ ] 캔버스 배경이 OS 다크 전환에 안 따라간다(라이트 고정).
